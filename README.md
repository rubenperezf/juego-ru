# Juego-Ru / Juego-Ru Game

---

## Español

Una colección de juegos educativos para practicar palabras en inglés y números.

### Juegos disponibles

#### Juego de Palabras
Practica el reconocimiento de palabras en inglés. Las palabras se muestran en pantalla en orden aleatorio y el jugador indica si las reconoce o no.

- Las palabras aparecen de colores distintos (negro o rojo)
- Hay un botón de audio (🔊) para escuchar la pronunciación de cada palabra
- Si marcas una palabra como incorrecta, vuelve a aparecer al final para practicarla de nuevo
- El juego termina cuando hayas completado todas las palabras correctamente

#### Números hasta 100
Aprende y practica los números del 1 al 100.

- El juego alterna aleatoriamente entre dos modos:
  - **Modo mostrar:** Se muestra el número y confirmas si lo reconoces
  - **Modo escribir:** El número está oculto y debes escribirlo tú mismo
- Debes responder correctamente para avanzar al siguiente número
- El juego termina al llegar al 100

### Cómo jugar
1. Abre `index.html` en tu navegador
2. Selecciona un juego
3. Presiona **PLAY** para comenzar
4. Usa ✓ si reconoces la palabra/número, o ✗ si no
5. Al terminar, presiona **PLAY AGAIN** para reiniciar

## English

A collection of educational games to practice English words and numbers.

### Available Games

#### Word Game
Practice recognizing English words. Words are displayed on screen in random order and the player marks whether they know them or not.

- Words appear in different colors (black or red)
- An audio button (🔊) lets you hear the pronunciation of each word
- If you mark a word as incorrect, it comes back at the end so you can practice it again
- The game ends when you've gone through all words correctly

#### Numbers up to 100
Learn and practice numbers from 1 to 100.

- The game randomly alternates between two modes:
  - **Show mode:** The number is displayed and you confirm whether you recognize it
  - **Type mode:** The number is hidden and you must type it yourself
- You must answer correctly to advance to the next number
- The game ends when you reach 100

### How to play
1. Open `index.html` in your browser
2. Select a game
3. Press **PLAY** to start
4. Use ✓ if you know the word/number, or ✗ if you don't
5. When finished, press **PLAY AGAIN** to restart
